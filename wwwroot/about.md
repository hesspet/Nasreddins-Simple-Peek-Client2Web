# Nasreddins Simple Peek Client 2 Web

Diese PWA steuert den BlePrompter per Web Bluetooth.

- Projekt: Nasreddins Simple Peek Client 2 Web
- Zweck: Steuerung des BlePrompter per Bluetooth Low Energy
- Plattform: Blazor WebAssembly PWA
- Auslieferung: GitHub Pages
- Zielbrowser: Android Chrome und iOS Bluefy
