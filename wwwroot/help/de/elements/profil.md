# Profil

Ein Profil speichert die vollständigen Einstellungen als lesbare JSON-Datei auf deinem Gerät. Der Dateiname ist gleichzeitig der Profilname und endet immer auf `.peekclient.json`.

Beim Laden prüft die App die Datei zuerst. Wenn Inhalt oder Werte nicht plausibel sind, wird eine Fehlermeldung angezeigt und die bisherige Konfiguration bleibt aktiv.
