# Einstellungen

Hier passt du Darstellung, Displayverhalten, Schlafzyklus, Videoquelle, Kamera-Bedienung und Audio-Spy-Mappings an. Änderungen werden automatisch auf diesem Gerät gespeichert.

Oben wird das aktuell geladene Profil angezeigt. Mit **Profil speichern** exportierst du die vollständigen Einstellungen als lesbare JSON-Datei im Format `{Profilname}.peekclient.json`. Mit **Profil laden** wählst du eine Profil-Datei aus; bei Fehlern bleibt die bisherige Konfiguration aktiv.
